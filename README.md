<h1 align="center">Swish 🏀</h1>

Swish 🏀 is a complete system for automating the publication of French basketball results. It includes a Node.js backend for headless image generation and an API for data processing. A script is used to obtain data and update the API. The React frontend provides an interface for building and visualizing content for social networks.
